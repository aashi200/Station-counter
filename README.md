# People Entry Counter

A simple web application that counts and records the number of people entering a location. The counter increases with each entry and allows saving previous counts.

## Features

* Increment counter to track entries
* Save previous counts
* Simple and interactive user interface
* Instant updates using JavaScript

## Tech Stack

* HTML
* CSS
* JavaScript

## How It Works

The application uses JavaScript to increment the counter whenever a user clicks the entry button. The current count is displayed on the screen and can be saved to keep a record of previous entries.

## How to Run

1. Clone or download this repository
2. Open the project folder
3. Open `index.html` in your browser
